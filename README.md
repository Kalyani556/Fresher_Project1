# CodezQuiz
Beginner level interactive quiz app on Python Programming which tracks your performance and shows your score.

Overview:
CodezQuiz is a small app to present questions based on Python programming, accept user answers, give instant feedback and show a summary score. It's simple Python code created using IDLE Shell.

Features:
1. It has MCQs as well as user accepted answers.
2. Final score
3. Remarks based on scores obtained
4. Interactive Quiz game

Technologies/Tools used:
1. IDLE (Python 3.12 64-bit)

Steps to install and run the project:
1.Install Python
2. Make sure “Add Python to PATH” is checked during installation.
3. Go to the folder where Codezquiz.py is located
4. or open the python file in IDLE
5. Select the quiz file (CodezQuiz.py)
6. Run the quiz inside the IDLE using Run module or press F5 on keyboard.

Instructions for testing:
1.Open CodezQuiz.py file in IDLE
2. Run the code by pressing run module or F5 on keyboard
3. Check whether the whole code runs without any syntax errors.
