Problem Statement:
Develop a simple Python-based quiz program that asks user basic questions about python concepts, accepts the answers, checks correctness and provides immediate feedback. The program should run in the Python IDLE shell. It uses basic input/output operations, keywords,loops etc. and help beginners test their understanding of python fundamentals.

Scope of the project:
1. A set of predefined questions related to python
2. Interactive input and output
3. Immediate responses
4. Case insensitive answer checking
5. Use of fundamental programming constructs
The defined scope ensures the project reamins simple, beginners friendly, and focuses on demonstrating fundamental python programming skills.

Target Users:
1. Beginner Python Users
2. School and college students
3. Self learners and online course participants

High level Features:
1. Interactive quiz interface
2. Predefined Question set
3. Case insensitive answer checking
4. Immediate feedback
5. Sequential question flow
6. Beginner friendly code structure
